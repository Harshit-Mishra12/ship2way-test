API Settings

The Ship2way API allows developers to build applications which interface with the Ship2way platform. The API can be used to automate many tasks including:

  • Managing Orders
  • Managing Shipments
  • Creating Shipping Labels
  • Retrieving Shipping Rates
  • and more!

Ship2way also has a separate 'Custom Store' integration option which allows you to import orders into your Ship2way account and receive shipping notifications when an order ships. This may be a more appropriate option if you simply want your store's orders to appear in Ship2way. You can find out more about our Custom Store integration here:

https://help.Ship2way.com/hc/en-us/articles/360025856192-Custom-Store-Development-Guide
